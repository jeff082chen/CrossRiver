# Cross River
人工智慧期末專題
  
# how to use
``` zsh
# clone the project
$ git clone git@github.com:jeff082chen/CrossRiver.git

$ cd CrossRiver

# arguments
# N: integer between 3~10
# M: integer between 0~2
# mode: 0 -> minimum price, 1 -> minimun time
$ python -m CrossRiver.main -N[3..10] -M[0..2] -mode[0..1]
```
